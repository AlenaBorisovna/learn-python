
оператор SELECT это запрос определнных операторв
состоит из 4 операторов:
SELECT - выбери
* - все колонки
from - из
employees - таблицы с именем employees

пример:
select name
from employees
limit 5;


# выводит 10 запросов
SELECT city
FROM employees;

# выводит 2 запроса
SELECT DISTINCT city
FROM employees;
